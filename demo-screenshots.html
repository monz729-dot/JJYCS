5. 실제 테스트 방법

🌐 브라우저에서 직접 테스트:

  1. 메인 화면 접속: http://localhost:5173
  2. 회원가입: "회원가입" 버튼 → 기업 선택 → 정보 입력
  3. 승인 대기 확인: 가입 완료 후 메시지 확인
  4. 어드민 접속: http://localhost:5173/admin/user-approval
  5. 승인 처리: 사용자 카드에서 승인/거절 버튼 클릭
  6. 로그인 테스트: 승인 후 로그인 가능 여부 확인
